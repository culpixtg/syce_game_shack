    This is the settings page, change settings for the site here!

    Mode
    Change the website's appearance by a dark and a light contrast.

    This changes the website's main theme color while using light or dark themed mode.

    Turning this off can improve faster loading times and help reduce network usage.

    This changes the website icon and title to help hide this among your tabs.

    This will keep the sidebar open and only close when you play a game or if you close it.
